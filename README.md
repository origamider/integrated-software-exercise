# Integrated Software Exercise

## セットアップ

### 必要なもの

- **GCC** (g++)

### macOS

Homebrew で GCC をインストール:

```bash
brew install gcc
```

### コンパイル & 実行

各フォルダ内の `.cpp` ファイルを `g++` でコンパイルして実行します。

```bash
cd phase1
g++ main.cpp
./a.out
```

### VS Code

C/C++ 拡張機能（ms-vscode.cpptools）をインストールすれば、IntelliSense が自動で動作します。
